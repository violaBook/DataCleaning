"过滤"模块说明书(filter-instructions):
---------------------------------------------------------------------------------------------------
module1-1[filter1_value.py]：						值
	功能: 给定一个范围对某一列满足要求的数据进行过滤
	-----
	演示: python filter1_value.py X_Force 0.888 0.999 './../test_data.pkl' 'pkl/filter1_value.pkl'
	参数说明: col_name min_data max_data
---------------------------------------------------------------------------------------------------
module1-2[filter2_type.py]:						值,字符串,日期
	功能: 给定一个类型对某一列满足要求的数据进行过滤
	-----
	演示: python filter2_type.py float './../new_data.pkl' 'pkl/filter2_type.pkl'
	参数说明: val_type
---------------------------------------------------------------------------------------------------
module1-3[filter3_integ.py]:						值，字符串
	功能: 对某一列按照数据完整性过滤 		
	-----
	演示: python filter3_integ.py X_Force './../new_data.pkl' 'pkl/filter3_integ.pkl'
	参数说明: col
---------------------------------------------------------------------------------------------------
module1-4[filter4_time.py]:						日期
	功能: 输入两个时间戳，查询时间戳范围内的数据	
	-----
	演示: python filter4_time.py end_time '2022-3-1 1:2:2' '2022-12-2 1:1:4' './../new_data.pkl' './pkl/filter4_time.pkl'
	参数说明: col_name low_time high_time
---------------------------------------------------------------------------------------------------
module1-5[filter5_duplicate.py]:					值，日期，字符串
	功能: 选中某一个属性值（某一列），去掉该列重复的行
	-----
	演示: python filter5_duplicate.py X_Force './../new_data.pkl' 'pkl/filter5_duplicate.pkl'
	参数说明: col_name 
---------------------------------------------------------------------------------------------------
module1-6[filter6_re.py]:						字符串
	功能: 对某一列的数据用正则表达式进行过滤后替换该数据
	-----
	演示: python filter6_re.py str_col 'b[abo]y_n\S*_[a-z]+r' './../new_data.pkl' 'pkl/filter6_re.pkl'
	参数说明: col_name  re_
---------------------------------------------------------------------------------------------------
